# MUSIC-RECOMMDATION-SYSTEM
This project uses Python to analyze music data, applying KMeans clustering, t-SNE, and PCA for visualization. It also features a Random Forest classifier with RFE for prediction, showcasing clustering and classification of musical attributes.
# Overview
This repository hosts a Python project focused on analyzing and clustering music data to discover underlying patterns in musical attributes across genres and years. The project applies several machine learning techniques such as KMeans clustering, t-SNE, and PCA for data visualization, along with a Random Forest classifier enhanced by Recursive Feature Elimination (RFE) for predictive modeling.

# Key Features
Data Exploration: Load and inspect comprehensive music datasets.
Clustering Analysis: Utilize KMeans to identify and visualize clusters within genre-specific data.
Dimensionality Reduction: Implement t-SNE and PCA to reduce feature space and visualize the distribution of music data.
Predictive Modeling: Develop a supervised learning model using a Random Forest classifier to predict music data clusters.
Tools and Libraries Used
Pandas for data manipulation
Scikit-learn for machine learning tasks
Plotly and Matplotlib for visualizing data
Seaborn for enhanced data visualization aesthetics
Usage
The scripts are designed to be straightforward and modifiable for different datasets or additional analyses. They provide a robust framework for anyone interested in music data analysis, from academic researchers to hobbyists.

# Project Structure
Data loading and initial exploration
Application of unsupervised learning techniques for clustering
Dimensionality reduction to visualize data clusters
Training a supervised classifier to predict cluster labels
Evaluation of model performance
# Conclusion
This project illustrates the power of machine learning in uncovering trends and patterns in complex datasets like those of music. By providing tools and methodologies for both unsupervised and supervised learning, it serves as a valuable resource for data scientists and enthusiasts alike.
